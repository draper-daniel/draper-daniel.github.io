CodeAcademy Cheatsheet Project: HTML/CSS

In this project I'm going to design a webpage in HTML and CSS to provide guidance for a topic I've learned so far. To do this I'm going to use tables in html, CSS typography and design rules, as well as both html and styling of lists.

Unordered Lists

HTML

Tag Name Formatted Tags Description Input Output
Unordered List <ul>
</ul>		 These are the tags that surround an unordered list.

<ul>

<li>First thing on the list</li>

<li>Second thing on the list</li>

</ul>
  • First thing on the list
  • Second thing on the list
List Item <li>
</li>		 These tags are used for each item in the unordered list.

CSS

The default CSS for unordered lists is bullet points.
This can be changed using the list-style-type: attribute.
There are some set examples shown in the table below, but we can also add in any string using list-style-type: "x"
Examples of this are also shown below.

Attribute Output
list-style-type: circle;
  • First thing on the list
  • Second thing on the list
list-style-type: square;
  • First thing on the list
  • Second thing on the list
list-style-type: "- ";
  • First thing on the list
  • Second thing on the list

[Notice how in the string bullet, there needs to be a space left after the bullet otherwise the list items begin immediately after the character.]

List Item <li>
</li>		 These tags are used for each item in an ordered list.

CSS

The default CSS for ordered lists is numbers starting at 1.
This can also be changed using the list-style-type: attribute.
There are some set examples shown in the table below.

Attribute Output
list-style-type: lower-roman;
  1. First thing on the list
  2. Second thing on the list
list-style-type: upper-alpha;
  1. First thing on the list
  2. Second thing on the list
list-style-type: korean-hanja-formal;
  1. First thing on the list
  2. Second thing on the list
